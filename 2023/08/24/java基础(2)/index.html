java面向对象

封装：把对象中的属性私有化，不允许外部直接访问对象内部的属性。但可以提供被外部访问的方法

继承：新的类可以调用父类的属性、方法，也可以有自己的属性、方法。

多态：建立在继承的基础上 对象在不同时刻表现出来的不同状态

继承下的访问权限

单继承 多实现 高内聚 低耦合

抽象类

抽象类abstract修饰，不允许实例化，抽象方法只有定义没有实现，为了强迫子类重写抽象方法

如果一个类中包含抽象方法，该类一定是抽象类，但是抽象类中没有抽象方法也可以

接口

interface 类对于接口继承 叫实现 implement 一个类可以实现很多个接口

jdk1.8之前不能有非抽象方法 jdk1.8之后可以用default定义非抽象方法

default void fun(){} 不需要所有实现类都重写这个方法 有需要的重写

父类与子类

父类 对象 = new 子类(); 以父类为基础，子类重写的方法替换父类的方法

子类 对象 = new 子类(); 以子类为基础，加上继承自父类的方法

第一次对类主动使用会调用静态代码块（只第一次），默认调用父类构造，再调子类构造 （大括号包裹，调用构造直接调用此代码块）

java中的工具类

String

String不可变 ==比较的是内存地址 equals比较内容

concat 拼接字符串 +

replace 替换单个字符

replaceAll替换所有

substring 截取 参数3，5 左闭右开 下标3到5

动态字符串拼接 Stringbuilder Stringbuffer

数组 Collection Map

ArrayList 初始可以传参数指定数组大小 调用无参构造初始长度为0 第一次添加时会设置长度为10

Arrays.copyof(arr,长度) 复制

Arrays.toString(arr) 转成字符串

Arrays.asList(1,2,3,4,5,6) 转成列表

Collections.addAll(list,内容) 添加

Collections.shuffle(list) 洗牌，乱序

ArrayList（数组）和Linkedlist（双向链表） 对比

首部添加LinkedList比ArrayList效率高

尾部添加两者效率相近

中间添加ArrayList性能优于LinkedList，LinkedList寻找中间位置耗时，每次从第一个向后找或者从最后一个向前找，都很难找到

查询时，LinkedList不善于用get方式取值，每次从第一个向后或最后一个向前

循环时使用迭代器或者增强for性能基本一致，如果循环下标，LinkedList性能差很多

set

HashSet 没有顺序，不限制内容

TreeSet 有顺序，限制同一类型内容，按照内容顺序排序

LinkedSet 按照添加顺序排序

map

HashMap key-value键值对 put存放 get取值

HashTable

泛型

规范了数据类型

包装类 8个数据类型封装出来8个引用类型

Byte Short Integer Long Float Double Character Boolean

Integer a = 10; Integer b = 10; a==b; //true

Integer a = 128; Integer b = 128; a==b; //false

缓存区-128到127指向同一个地址 超过之后会有新的对象

a.intValue() == b.intValue(); //true 拆箱
