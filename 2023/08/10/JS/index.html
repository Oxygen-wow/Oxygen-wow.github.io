javascript

JS代码写在head或body里

变量 let 变量名 = 常量 const 变量名 = 常量 var 变量名 = 常量

数据类型 number string boolean(true false) function函数 undefined未定义

函数 console.dir(变量或常量)控制台打印 alert 弹出提示框 typeof查看变量数据类型 confirm确认取消弹窗 prompt输入框弹窗 parseInt取数字

object 数组 [ ] length长度 push添加 splice(下标，个数，替换内容)

JSON {属性名：属性值，属性名：属性值} 取值 . 或 变量名[属性名]

循环 while for do while 停止循环 continue停止本次 break结束整个

setTimeout定时器 执行一次 setInterval循环执行

DOM

Document Object Model 操作HTML

获取方法 旧版 document.getElementById()
document.getElementsByClassName()
document.getElementsByTagName()
document.getElementsByName()

新版 document.querySelector() id #a class .a 有重复获取第一个 document.querySelectorAll() 获取数组

innerHTML 包括标签的html

innerText 只包括文字

事件 触发函数运行

鼠标 onclick鼠标点击 onmousedown鼠标按下 onmouseup鼠标抬起 onmousemove鼠标移动事件 onmouseleave鼠标离开事件(不传递子元素) onmouseout鼠标离开事件(会传递到子元素) onmouseover鼠标移上事件(传递到子元素) onmouseenter(不会传递到子元素)

键盘 onkeydown键盘按下 onkeyup键盘抬起 onkeypress键盘按下 只有在输入框产生字符才会触发 onblur失去焦点 onfocus获取焦点 onchange值有变化失去焦点

日期时间对象 new Date() 获取系统日期和时间 getYear() 获取当前月份和1990之间的年份差 getFullYear() 获取当前年份 getMonth() 获取月份 getDate()获取日期 getDay()获取星期 getHours()获取小时 getMinutes()分钟 getSeconds()秒 getMillseconds()获取毫秒 getTime()获取时间戳

jQuery

js库，位操作dom提供了统一的api

$() 获取dom 参数选择器 穿function入口函数 当页面中的标签加载完成之后运行该函数

事件 和js中的事件一致 去掉on function作为参数来传递 虚拟dom放进实体dom形成子元素显示在页面上 虚拟dom.appendTo(实体dom) 虚拟dom.prependTo(实体dom) 实体dom.append(虚拟dom) 实体dom.prepend(虚拟dom)

函数 height()获取高度 height(参数)设置高度 width()获取宽度 width(参数)设置宽度 css(参数传json)设置样式 offset获取距左距顶边距，返回json index()返回同级元素下标 eq(索引)获取具体下标 fadeIn淡入 fadeOut淡出 siblings(选择器)
