HTML+CSS

|字数总计:1.6k|阅读时长:5分钟|阅读量:|评论数:

HTML

1.<标签名> 开头

2.html标签 内容写在html里 一个页面只能有一个html

3.head标签

meta

charset 编码格式

name=keywords 关键词

description 描述信息

4.body

div 块区域 设置了宽度和高度都会有效果，如果不设置宽度占100%，如果不设置高度随内容填充

span 行区域 设置了高度宽度无效

h1到h6 标题标签

列表标签 ol li 有序列表 type属性 默认1按照数字排序 A按照大写字母排序 a按照小写字母排序 I按照大写罗马数字排序 i按照小写罗马数字排序

列表标签 ul li 无序列表 ul是父元素 li为ul中的父元素

p标签 不能包裹其他标签 只能

a标签 超链接和锚点 设置href target 跳转目标 在当前页面跳转 _self默认 _blank 新的页面打开 锚点：#+name值 跳转至name值的位置 快速定位

strong 加粗

br 换行

em 斜体

语义化标签 header头部 nav导航 main主要内容 section块 aside侧边栏 footer页脚

表单 form标签 input(type默认text 一般输入框 password密码 date日期 time时间 number数字 file文件选择 color颜色 hidden隐藏内容 radio 单选框需要有name值，name值相同的为一组，一组中只能选一个 checkbox多选框 button按钮已废弃 reset重置按钮已废弃 submit提交按钮已废弃)

button标签 button reset submit

textarea区域框 rows显示行数

select选择 搭配option下拉框

datalist下拉框 搭配option和list

table 表格 thread表头tr th tbody表身 tr td td中使用rowspan合并行 colspan合并列 border-collapse: collapse;合并单元格边框线

iframe框架页 嵌套页面 设置name 让a标签套target

CSS

选择器

#匹配id值(一个标签只能有一个id值) .匹配class值(一个表标签可以有多个class值) 元素选择器 内联样式>id>class>元素选择器

伪类选择器 hover鼠标放上后变色 neh-child(索引) nth-last-child(索引)

层级选择器 父类>子类 只匹配子类 父类 子类 不限制层级

样式 盒子模型

width宽度 height高度 background背景颜色 红绿蓝 #000000 宽高 写法 属性名:属性值 盒子模型实际大小 宽度:width+左右内边距+左右框线 高度:height+上下内边距+上下框线 border-sizing:content-box默认content border-box内边距和框线往里走

外边距margin(1个值上下左右 2个值上下 左右 3个值 上 左右 下 4个值 上右下左) margin-left和margin-right都设置为auto会相对居中

内边距 padding 1个值 上下左右 2个值 上下 左右 3个值 上 左右 下 4个值 上右下左

边框线 border四个框线 solid实线 dashed线虚线 dotted点虚线 inset凹陷感 outset凸起感

display block将元素转换为块元素 inline将元素转换为行元素 inline-block 将元素转换为行内块

转义 空格 &nbsp；& &amp；< &lt；> &gt；

浮动 脱离文档流浮动可并排 清除浮动 clear left right both 必须写在有浮动的同级元素，清除浮动保持浮动位置

定位 不改别人，只改自己 position static默认固定 relative相对固定不脱离文档流 absolute绝对定位脱离文档流，起点位置先看父元素有无定位，有则参照父元素，无则向上看 fixed固定定位脱离文档流屏幕 top left right bottom 覆盖规则有定位盖着没定位的

line-height 行高

text-align 文字位置对齐方式 left right center

font-size px em rem 使用rem批量修改字体大小 html{ font-size：16px font-family: ‘微软雅黑’}

长度单位 px %参考父元素 em参考父元素的font-size rem参考html的font-size vh参考浏览器的高度100vh vw参考浏览器宽度100vw

text-decoration 下划线

弹性模型 区域划分 display：flex flex-direction父元素决定分配方向 row水平分配 column垂直分配 row-reverse水平翻转 column-reverse垂直翻转 align-items如果水平分配则垂直对齐，如果垂直分配则水平对齐 flex-start开始位置对齐 flex-end结束位置对齐 center居中对齐 justify-content如果水平分水平对齐，垂直分垂直对齐 space-between空隙

img 行内块 src路径 设置大小通过css宽度和高度 第二个直接在标签上设置

background-color背景色 background-image背景图url(图片路径)

background-repeat平铺 repeat-x repeat repeat-y no-repeat background-position两个值，水平位置，垂直位置，center居中 background-size 两个值 宽度和高度 contain小比例改变 cover大比例改变

伪元素 选择器::before 选择器::after

opacity透明度

flex-wrap 默认不换行nowrap 超出后换行wrap

阴影效果 块加阴影 box-shadow 文字加阴影 text-shadow

过渡效果 transition 过渡时间

圆角 border-radius 值为半径 1个值四个角 2个值左上右下和右上左下 4个值顺时针一圈

overflow内容超出hidden隐藏 配合text-overflow: ellipsis和white-space:nowrap 效果…

font-weight字体粗细 100-900 正常400 bold等于700(h1-h6)

transform转换 缩放scale(0到1之间) rotate(数值deg)旋转

3D效果 transform-style:preserve-3D 需要在父元素上加镜头的距离 perspective 800-1200px 搭配transform旋转rotateX(deg)沿x轴 rotateY()绕Y轴 rotateZ()绕Z轴 平移translateX(px) transform-origin三个参数 元素变换的水平垂直z轴的中心点

倒影 -webit-box-relect只有webit浏览器支持 above上方 below下方 left左边 right右边 可加间距

渐变 background 线性渐变 -webit-linear-gradient(deg,颜色) 镜像渐变-webkit-radial-gradient();四个参数 渐变点 渐变范围

动画 @keyframes 动画名 {} animation 过渡时间 速度曲线 ease慢快慢 ease-in慢快慢 ease-out快慢 ease-in-out慢快慢 linear全程匀速 延迟时间 次数 number infinite无限次 逆序 alternate偶数次逆序 停留位置 动画结束后停留最后一帧forwads

动画效果要想支持不同浏览器 -webkit-animation -moz-animation -o-animation @-webkit-keyframes @-moz-keyframes @-o-keyframes
